团队介绍

创始成员、顾问，与专家组。

ADPS 汇集了 Agent 架构、软件工程、开源基础设施和企业系统落地领域的研究者与实践者。 团队共同维护模式目录，发布参考资料，并用可运行系统和一线案例持续检验这套模式语言。

创始成员

黄佳（Jia Huang） 简历 PDF

Agent 架构 · 模式语言 · 工程教育

新加坡 A*STAR AI 研究员，Manning 《Designing AI Agents》 作者。研究重点是用成熟的软件工程方法设计生产级 Agent 系统，包括明确的职责边界、 执行拓扑、验证机制和可复用模式。同时著有《Agent 设计模式》，并主讲极客时间 《Claude Code 工程化实战》课程。

代表工作：

  • 共同提出并公开双轴框架：认知功能 × 执行拓扑。
  • 持续维护 ADPS 模式目录与选型方法，使其成为可讨论、可评审的架构语言。
  • 把书籍、课程、企业案例和可运行参考实现连接成一套工程词汇。

茹炳晟（Bingsheng Ru）

软件工程 · 智能体系统 · 研发效能

腾讯研究院产业研究顾问，中国计算机学会 CCF TF 多智能体系统工程 SIG 副主席， 复旦大学 Codewisdom 首席技术专家。国内软件工程、智能体系统、研发效能、 AI 赋能研发领域的重要实践者和传播者。

长期作为技术峰会出品人、联席主席、Keynote 演讲嘉宾与技术图书作者/译者活跃在社区中—— 把 Agent 设计的讨论连接到更广阔的中国软件工程共同体。

在 ADPS 的工作：软件工程方法、智能体系统工程、研发效能与复杂系统评估。

代表贡献：

  • 把 Agent 放回真实软件研发现场，而不是停留在玩具题和 demo。
  • 从研发效能、软件复杂性、组织协作和工程评测角度，为 Agent 设计模式提供现实边界。
  • 连接 CodeWisdom、腾讯和中国软件工程社区，让 ADPS 不只是 AI 圈的概念，而是软件工程共同体的延伸。

开源基础设施 · 系统集成 · 工作流

Apache 软件基金会长期贡献者，Apache Camel 社区重要成员，开源社区活跃实践者； 公开资料显示，他参与 DeerFlow 2.0 相关公开活动，并在社交资料中标注 DeerFlow developer。

梁博（Bo Liang）

企业 SaaS · 金融服务 · 生产系统

上海东方屹腾科技有限公司副总裁，创办期股东与合伙人，负责技术和产品。东方屹腾核心团队来自银行 互联网渠道领域，聚焦银行泛金融生态场景 SaaS 服务；其薪享通 SaaS 服务矩阵已与 20+ 银行 建立联合运营体系，服务 5 万+ 企业客户，并正在把 SaaS 产品矩阵全面转型为 Agents 体系。

在 ADPS 的工作：企业部署证据，以及金融 SaaS 向 Agent 系统转型的一线实践。

代表贡献：

  • 把 ADPS 的模式语言放进银行泛金融、HR SaaS、薪酬、考勤、个税、薪资代发等复杂业务流程里验证。
  • 提供从 SaaS 到 Agents 的转型场景：不是"加一个聊天入口"，而是把企业服务流程改造成可预算、可观测、可授权、可复盘的智能系统。
  • 作为实战型产品与技术伙伴，持续把真实企业问题、架构反馈和产品边界带回框架。
